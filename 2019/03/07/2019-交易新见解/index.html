2019-交易新见解

Mar 7, 2019 | 金融 | 阅读

震荡是比趋势简单的,震荡只需要判断一个大框架即可操作,趋势的难点在于判断,判断错误那么就是震荡

MACD级别性回归(技术点)已无用

引用角少的话

MACD级别性回归这个技术，历经了5,6年。正式宣告这个技术再无之前的高命中率和高胜率了。被时代淘汰。衰败于2018年4月25日，死于2019年3月7日。世间再无MACD级别性回归技术，世间再无角式技术分析。

6年技术体系进化历程： 强势底分型 15分钟回踩操作 配合MACD 0轴和60线 → 多空头震荡中枢的分类 → 逆推法+大小级别联立的确认体系 → 市场的多维度共振 → 背驰无用论→ 高低点系统+均线体系 → 多级别大趋势线的走势分类 → 和字诀→ 一切成本来自于市场波动 → 个股先于指数的运行特征 → 彼之静态 吾之动态

快过年了的那几天，我叫了几位还不错的老学员们凑一起帮他们上了一课，也算是对我那几个礼拜思考的一次语言上的汇总，在那会儿我把历年寻找的规律，结构也好，均线也好，化繁为简，命名为特征。 这是什么意思呢？ 比如 微博的老朋友们特别喜欢看奔走型中枢的操作，把奔走型中枢当作一个十分重要的技术指引，那么会导致什么问题呢，奔走型中枢向上突破就想做多，奔走型中枢向下突破就想做空。当然了，许多人会联立。但是许多人的联立跟我的联立并不同，我的联立是基于对走势的认识之联立，许多人的联立是之于自己喜好之联立，会挑各个级别自己喜欢看到的，希望看到的多头信号点作为联立，头头是道，称之为多级别联立。方法没错，人有问题。类似奔走型中枢也是这么个问题，以奔走型中枢为喜好，自然能联立出一堆东西来。奔走型特征不是核心，走势的价格才是核心。 既然走势的价格才是核心， 那么他出现的何种特征就不是问题的关键了。

再比如：确认这个概念。 许多人认为的确认就是等于反向中枢破位或者三笔破位。认为确认的出现就是 反向的破位。那仅仅是价位上的破位，不是价格上的破位，所以也不是核心，更有甚者，越是强势的行情越找不到确认，这又是为何呢？ 不是您找不到确认，而是您把确认钉死在了某些固定的特征上了，这些固定的特征不出现，自然不存在确认。那是为何呢？ 原因还是对走势的价格何时走强走弱的核心不理解，那么他是否出现固定的特征破位就不是问题的关键了。

有上面两个角度，自然就可以化繁为简了，是不是奔走型重要吗 不重要。上不上250 重要吗 不重要。 放不放量重要吗 不重要。 低点抬高重要吗，不重要。统统不重要。 那什么才是重要的呢？走势价格的走强走弱的过程中伴随着这些特征的出现，才重要。价格有三个核心： 1 级别 2 空间 3 动态。 有这三样东西在，世间特征千变万化，不仅仅是我总结的那些，别的也都可以海纳百川。好的，点到为止，仅提供思路，各位可以结合自己的困惑想想吧。
