部署 wildfly


运行 standalone.sh

访问 http://127.0.0.1:9990 , 能访问标识部署成功,接着配置用户

运行 add-user.sh

What type of user do you wish to add?
a) Management User (mgmt-users.properties) (这个是配置WildFly服务器的用户)
b) Application User (application-users.properties) (这个是配置部署在WildFly服务器app[比如即将要在WildFly部署的Drools Workbench应用]的用户)

接着自行配置

将下载的 Drools Workbench war 改名改为 kie-wb.war (这里可以不改,或者改成自己喜欢的名字)

然后部署上去即可.
