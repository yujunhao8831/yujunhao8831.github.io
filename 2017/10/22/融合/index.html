    • 跌的漂亮

    当然如果说它没回调在预想的50%上就直接上去了,那没话说
    只是说,如果回调在50%上再上去,那么之后的上涨空间会很好

    二买

    打破高低点后,回抽在0轴上的二买,涨破之前高点后并且突破250日线的二买是很强的,胜率很高.

    找背驰

    等一直在 MACD0 轴以下晃荡的走势结构何时完美，然后才去找背驰

    • 从次级别中精确

    大级别回调，怎么精确进入。就拿现在60分钟内背来说，看去柱子缩短，股价新低了只能叫内背段，已经完全可以一个强力打压而柱子继续伸长。
    宏观上看叫内背，具体细节还得从小级别上来，只有小级别的走势结构完成背驰了，才能确立大级别内背的确立。

    三卖后的背驰开多,三买后的背驰开空

    三套技术的总结

    1. 均线 多级别

    • 别轻易只用一个级别上的均线，不仅仅走势才是多级别联立，均线也是需要的
    • 均线的意义在于股性和趋势的划分，以及关注均线附近形成的结构，而不是把均线当成支撑和阻力去弄
      • 均线需要配合的东西有两个
        • MACD
        • 均线附近的走势结构，学会用笔操作

    2. 高低点在走势结构中的运用

    • 高低点在盘整走势中是没有任何作用的，但是可以通过整体看看盘整结构能量的移动
    • 要学会用线段走势分析，不要一辈子拘泥在中枢中，高低点运用熟练的完全可以忘却中枢
    • 不要试图用背驰去判断一根线段当下的结束，没人可以判断所有线段的技术
      这个技术的精髓点一句很简单的话，上升线段中有效的是回抽的低点位置，下降线段中有效的
      是反弹高点的位置。道理很简单，是否产生另一根线段，线段去破坏线段。

    3. MACD

    • 最关键的自然是他的0轴

    根据MACD的0轴为界限，划分为多头中枢，空头中枢，和中阴中枢。多头和空头这两类中枢需要根据不破不立的原则。

    逆推法

    非标准趋势中,顺势的那一段都是高低点标准延伸的,那么背驰只保证一笔的的调整或者回抽.这里的背驰没多大用

    什么是结构性背驰?

    打破高低点格局,然后再背驰,就是结构性背驰. (这里的背驰是可以介入的)

    如下图 :

    A,B点的背驰,如果此时做空,止损设置在哪里呢?
    如果B点的背驰回抽没有打破高低点格局,后续走势还是可以继续构造一个中枢后,再上涨.

    目前看到的走势是,B点的背驰回抽打破高低点格局,但是还是回抽在0轴之上,很明显的回踩在60线上.
    之后的C点的背驰就是结构性背驰,具体精确的出场点就从小级别确定就可以了

    一种技术是不能应对所有的走势

    比如奔走型这个分析，虽然很好用，但不是时时刻刻都会出现奔走型的.奔走型的走势结构，还要注意简单三笔就线段结束的情况.
    那么这个自然需要第三笔的不背驰，以及第四笔的回抽必须得在0轴上或者下，起码也得粘着0轴来宣他的强势来确认真正的奔走型中枢的出现

    • 同样一个奔走型中枢，如果他的第三笔的回抽在0轴上，同样是低点抬高和高点抬高额一个中枢，但是一个第三笔不在0轴上和在0轴上这个小小的区分可以让行情相差许多

    不需要做到全部的行情，你可以区分出自己认为划算的行情， 下重仓位，那么你的资金翻滚率肯定远大于你行情都做的。

    结构的分类

    把走势只要分成三种，一个是无方向的盘整。一个是标准趋势。一个是非标准趋势.

    无方向的盘整

    那么就是一根能量平衡点就行了，能量平衡线的划分:该盘整的中枢的密集区，划一个二分之一的线过来就行了

    标准趋势

    他走势的结束非常简单，趋势背驰就行了.

    但是
    趋势结束以后会跟个反趋势吗?
    标准趋势背驰了之后，可以扩展后直接向上吗?
    可以走个无方向的盘整，根据能量平衡线再选择跟上向上走势还是向下走吧?

    趋势背驰虽然比较靠谱，但是也不至于可以靠谱的一定引发一个反趋势

    所以背驰只是一方面，后续跟扩展，也是可以根据高低点的.

    标准趋势后进行无论扩展还是反趋势，都是意味着从标准向非标准转变，高低点马上可以用

    非标准趋势

    可以完完全全的根据高低点来操作

    这里就不需要去划分什么中枢之类的，只需要留意里面几个简单但是实际的中枢就行了，空头中枢，多头中枢，奔走型中枢.

    • 盘整背驰
      无方向盘整的震荡，造成的新高新低的背驰叫做盘整背驰。
      在趋势背驰后，构造的第一个盘整，上破，或者下破中枢的背驰也是盘整背驰，这个要留意出现三买。盘整背驰就是可以分成这两块。

    • 趋势背驰 非标准趋势背驰
      这两个背驰都有个共性，就是肯定是有结构出现，然后背驰，最后才会有走势终完美的说法
      非标准性趋势背驰就可以分成结构性背驰
      结构性背驰

      • 是打破高低点格局之后的向上新高或者向下新低引发的背驰
      • 三买和三卖之后的背驰

    这两种都是通俗的结构性背驰在非标准趋势里的表现形式

    背驰无用论

    结构需要确认,确认的真实性在于破位

    缠所认为的背驰，是基于什么?基于市场结构

    真正靠谱的是，大级别的背驰段，小级别结构确认背驰成功.

    那么在 0 轴下，且没发生任何结构式信号点的背驰，是不是可以当做是指标的钝化呢?

    市场走单边市的时候，指标会钝化。这里背驰不要去管他，继续持有，更不会去抄顶抄底了，我相信百分之 90 的人会被市场急杀秒盘，都是在做顺势 0 轴上或者 0 轴下指标背驰的时候，然后进去一个不多的回抽，直接顺着原来方向拉直线思密达了。这也是为什么我开篇就会说，做一个极致的右侧交易者。那么自然不会存在这些情况。这里背驰给出的基调就是，他只是个警示点，他可以用于平仓见好就收，但是单纯的背驰在开仓方面不存在任何意义。

    那么怎么去过滤这些背驰呢?
    放大级别

    当市场出现这些指标背驰的时候，放大你软件的周期图，根据逆推法来说，小级别的线段趋势，大级别走的是一笔，那么你完全可以根据大级别来进行分析。
    背驰的是否有用是建立在结构上的,在没有结构的前提下，往往保证的是一笔回抽进行技术指标的修复。

    • 在构造空头中枢，多头中枢的背驰，是没有用的，这个一定要满足，这些中枢的向下破坏，才有可能走势方向改变
    • 单边走势背驰无用，走势终完美需要打破高低点格局，结构性背驰以及不新高或者不新低的确认
    • 高低点延伸的背驰，向上走势，低点一直抬高，背驰只保证一笔回抽，格局不打破，线段依旧延伸。依旧需要打破高低点格局，结构性背驰或者不新高之后一个结构的确认
    • 大级别趋势下的背驰
    • 大级别趋势下,小级别的背驰

    指标背驰无用，指标背驰是警示点，不可用于开仓。背驰基于结构有效，背驰的确认需要更小级别的结构确认，在某个级别的单边市场中，MACD作废，调大看图周期。

    规矩

    先看背驰，再看确认，确认总要下的吧，那么在关键点位附近开始挂单.
    只等待你的机会，不去瞎猜，在关键点位才成交，不给拉倒，避免不必要的折损.

    • A 代表大级别向上趋势 a 代表小级别向上趋势
    • B 代表大级别向下趋势，b 代表小级别向下趋势
    • C 代表大级别震荡。 c 代表小级别震荡
    • 0 代表 0 轴或者 60 线。
      那么这里就会有许多组合方式来测量

    当下行情为 :
    A,a 做空? 找死
    A,b 做空? 短期可以试试(55开)?这么多品种为什么要做这种胜算不大的呢?所以还是不做这种
    B,b 做空? 很好,大势所趋

    那么什么行情又不会死利润又多的。毫无疑问肯定是趋势配合震荡。因为震荡即是盘整.

    盘整的变盘，利润是最多且最牢固的

    A,c 并且不上0轴可以找到向下变盘
    B,c c 处于 B 级别中的 0 轴下时，往往是中继，利润也是巨大的

    市场行为

    当对应的两个大小级别，如 60 分钟和 15 分时，60 分钟出绿柱时，15 分钟满足黄白线必下 0 轴。那么当 60 分钟出绿柱时，15 分黄白线在 0 轴上，那么肯定会有一波黄白线下 0 轴的过程

    以 15 分钟为例，你找不到一个上涨趋势在 15 分钟 60 线下的，你也找不到一个下跌趋势在 15 分钟 60 线上的。
    那么好，如果一只股票，在周线日线 60 分钟 15 分钟 都在 60 线上，是不是市场行为趋于一致

    市场情绪

    级别越大，那么这个技术点越稳定
    大级别的交易，技术不重要，正确的理念很重要，以及一颗大心脏能够无视市场噪音

    大的级别出现背了又背的情况，概率很小，而越小的级别，比如一分钟，出现背了又背的情况简直是烂大街.
